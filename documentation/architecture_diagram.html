YOLO26 + ASIO Optimized

The primary proposed setup. Uses the fine-tuned YOLO26-seg model with dynamic Swarm hyperparameter search and Dice-weighted server aggregation.

  • Model Initialization: Pre-trained YOLO26-seg (BraTS best.pt)
  • Aggregation Strategy: ASIO Dice-Weighted
  • Hyperparameter Tuning: Dynamic ASIO Optimizer active
  • Warmup: Disabled (trains instantly with searched lr)

YOLO26 Baseline Baseline

The standard YOLO26 comparison setup. Uses standard sample-weighted FedAvg parameter aggregation and standard fixed hyperparameters.

  • Model Initialization: Pre-trained YOLO26-seg (BraTS best.pt)
  • Aggregation Strategy: Standard FedAvg
  • Hyperparameters: Fixed (lr0=0.01, conf=0.25, batch=8)
  • Warmup: Disabled

YOLO12 + ASIO Optimized

The lightweight YOLO12-seg setup with the ASIO optimizer. Starts from scratch on BraTS, utilizing dynamic restricted bounds and lenient validation conf thresholds.

  • Model Initialization: Raw COCO weights (3-class adapted)
  • Aggregation Strategy: ASIO Dice-Weighted
  • ASIO lr0 Bounded: restricted to (1e-4, 5e-3) for convergence
  • Lenient Val Confidence: Dynamic, bounded to (0.05, 0.25)

YOLO12 Baseline Baseline

Standard YOLO12 configuration. Serves as the ultimate baseline comparison, using raw COCO weights and standard sample-weighted FedAvg.

  • Model Initialization: Raw COCO weights (3-class adapted)
  • Aggregation Strategy: Standard FedAvg
  • Stable Fixed Hyperparameters: (lr0=0.001, conf=0.15)
  • Warmup: Disabled
